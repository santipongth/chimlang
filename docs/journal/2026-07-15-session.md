Findings ที่เตรียมรายงานผู้ใช้:
1. API บาง path ไม่มี lower bound ให้ `agents`: `/runs` ใช้ `n = min(body.agents, engine.max_agents)`, `/gallery/share` ใช้ `_run_dashboard(..., body.agents)` และเก็บ `agents=min(body.agents, cap)`, `/watchlists` เก็บ `agents=min(body.agents, cap)` แล้ว `default_runner()` ใช้ค่าที่เก็บต่อ ทำให้ค่าติดลบ/ศูนย์จาก API ตรงผ่านเข้า simulation ได้ ต่างจาก `/jobs/whatif` ที่ใช้ `min(max(req.agents, 10), cap)` แล้ว
2. `tests/test_phase6.py` ทำ PostgreSQL readiness check ตอน import ด้วย `psycopg.connect(DSN)` ไม่มี `connect_timeout`; เมื่อ dev stack ไม่เปิด `pytest`/collect ค้างก่อน skip ได้ เป็นสาเหตุที่ full suite timeout ใน session นี้
