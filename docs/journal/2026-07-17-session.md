# Session journal — 17 ก.ค. 2569

## Codex — Phase 9 M1 Pilot-ready Trust Foundation

### สิ่งที่ผู้ใช้สั่งและมติที่ถือเป็นฐาน

ผู้ใช้ส่ง roadmap Phase 9 ที่วิเคราะห์จาก codebase/research แล้วและสั่งให้ implement ใน fresh context.
Roadmap กำหนด 8–12 สัปดาห์ แบ่ง M1–M3 และระบุชัดว่า M1 เป็น trust gate: เมื่อรายงาน migration,
concurrency/SSRF tests และ UX screenshots แล้วต้องหยุดรอมติ ห้ามเดินเข้า M2 เอง. จึงจำกัด session นี้
เฉพาะ P9-M1 และบันทึกสถาปัตยกรรมเป็น ADR-0014 ก่อนแก้ implementation.

เหตุผลหลักที่อธิบายระหว่างทำงาน: ความเสี่ยง pilot ปัจจุบันอยู่ที่ trust contract, cancellation race,
asynchronous workflow, SSRF และ navigation มากกว่าการเพิ่ม simulation engine. คำว่า replay ต้องไม่เกินจริง:
provider pin seed ได้ best-effort เท่านั้น ตัว freeze ที่ตรวจได้คือ input/result snapshot + manifest hash.

### งานที่ทำ

1. เปิด Phase 9 ใน `CLAUDE.md`/`AGENTS.md`, สร้าง `docs/PHASE9-BRIEF.md` และ ADR-0014.
2. เพิ่ม `core/run_manifest.py`: frozen Pydantic contracts, canonical JSON/SHA-256, completeness reasons,
   request/config/artifact hashes, code/prompt/model/pricing/governance snapshots และ append-only store.
3. เพิ่ม migration `2026-07-17-run-manifests-v1`; legacy backfill เป็น incomplete โดยไม่เดา provenance.
4. เปลี่ยน `RunStore` terminal transitions เป็น compare-and-set; canceled/error/complete เขียนทับกันไม่ได้,
   posts ถูกปฏิเสธหลัง terminal และ worker เช็ก cancellation ก่อนทุกจุดเขียนสำคัญ.
5. เปลี่ยน async contract เป็น 202 + `Idempotency-Key` และ run-centric URLs; เพิ่ม manifest/snapshot API,
   frozen/latest rerun และ stored-snapshot JSON/PDF export พร้อม watermark/manifest hash.
6. สร้าง `SafeOutboundFetcher`: A/AAAA/global-only, DNS pinning, redirect revalidation, downgrade/type/
   compressed/decompressed limits; ต่อเข้า sources และ RSS News Desk.
7. เปลี่ยน frontend router เขียนเองเป็น React Router HashRouter; เพิ่ม typed routes/404/run-gallery-
   experiment deep links, mobile drawer accessible, running timeline/reconnect/cancel/error recovery.
8. New Run เลิก poll สูงสุด 240 วินาทีและเปิด Run Detail ทันทีหลัง 202; Run Detail แยก stored snapshot,
   frozen rerun และ latest rerun ด้วยข้อความไม่อ้าง exact replay.
9. เพิ่ม named Pydantic response/error contracts แล้ว regenerate `openapi.generated.ts`.

  และแก้เฉพาะ workspace; ไม่ใช้ shell write trick.

  อัปเดตเป็น 6.30.4 และยืนยัน production audit 0 แทนการยอมรับ vulnerability.

  index 92.67 kB (+13.0%) และ aggregate initial dependencies ยังต่ำกว่า gate; graph/charts ยัง lazy.

  normalize request กลับสู่ public bound ก่อน enqueue และเพิ่ม regression test.

  `Content-Length` ออกจาก limit check เพื่อคง error taxonomy ที่ถูกต้อง.
- Playwright bilingual test เก็บ locator aria-label ภาษาไทยแล้ว label เปลี่ยนหลังสลับ EN; แก้ testให้ค้น
  label ใหม่ และเพิ่ม focus restore assertion ซึ่งผลักให้ drawer implementation มี focus trap/restore จริง.
