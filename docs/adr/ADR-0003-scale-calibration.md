# ADR-0003: Calibration พารามิเตอร์ scenario ที่ scale ≥ 100 agents + broadcast mode ใน engine

- สถานะ: Accepted (6 ก.ค. 2026 — ผู้ใช้สั่ง re-calibrate หลังพบ finding จากการวัด scale)
- บริบท: หลังยกเลิก cap 10 → 1,000 พบว่า what-if delta หดจาก −16.5% (n=10) เหลือ −1.2% (n=1,000)

## การวินิจฉัย (ตัวเลขวัดจริงทั้งหมด, กลไกล้วน $0)

1. **ข่าวลือจาก seeder คนเดียวไปไม่ถึงประชากรใหญ่ใน 30 rounds** — penetration:
   n=10 → 92%, n=100 → 47%, n=1,000 → 8% เพราะช่องทางกลุ่มปิด/เพื่อนบ้านมี reach
   คงที่ต่อคน (สมจริง — เมืองใหญ่ใช้เวลาหลาย generation กว่าจะอิ่มตัว) แต่หน้าต่าง 30 rounds เท่าเดิม
2. **คำชี้แจงทางการก็ไหลจาก "คนเดียว"** — ผิดธรรมชาติ: ของจริงแถลงผ่านสื่อถึงมวลชนพร้อมกัน

## การตัดสินใจ

สำหรับ scenario "ข่าวลือแรงระดับเมือง + การชี้แจงทางการ" ที่ scale ≥ 100:

| พารามิเตอร์ | ค่า calibrate | เหตุผล |
|---|---|---|
| ข่าวลือตั้งต้น | `preseed` 10% ของประชากร | ข่าวลือที่สังเกตได้จริงไม่ได้เริ่มจาก 1 คนในหน้าต่างสังเกต |
| หน้าต่างเวลา | 60 rounds (≈ 10 วัน ที่ 4 ชม./round) | penetration 76–94% เทียบเคียง n=10 |
| คำชี้แจงทางการ | `broadcast_share=0.20` (feature ใหม่ใน engine) | แถลงข่าวผ่านสื่อถึง ~20% ทันที ไม่ใช่ diffuse จากคนเดียว (สอดคล้อง FAB-03) |

**ผลหลัง calibrate**: delta −15.0% (n=100) / −15.1% (n=1,000) — scale-invariant และสอดคล้อง n=10 เดิม

## ผลกระทบ

- `Message.broadcast_share` เพิ่มใน engine (default 0 = พฤติกรรมเดิมทุก run ก่อนหน้า ไม่กระทบ reproducibility)
- ผลเชิงปริมาณที่รันด้วย seeder เดียว ณ scale > 10 ก่อน ADR นี้ ให้ถือว่า underestimate ผลของ intervention
- FAB-01 benchmark (isolated channel, n=10) ไม่ถูกแตะ — ฟิสิกส์ช่องทางเดิมทั้งหมด
- ทบทวนอีกครั้งเมื่อ calibrate segments กับข้อมูลสำรวจจริง (คิว P3-Q รอแหล่งข้อมูลจากผู้ใช้)
