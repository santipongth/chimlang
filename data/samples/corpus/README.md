# corpus — เอกสารทดสอบสำหรับ ingestion pipeline (M2)

วางไฟล์ข่าว/เอกสาร `.md` หรือ `.txt` ระบุวันที่ในชื่อไฟล์ เช่น `2026-05-12-หัวข้อ.md`

## ⚠️ สถานะของไฟล์ชุดนี้

ไฟล์ทั้ง 12 ไฟล์ในโฟลเดอร์นี้เป็น **เอกสารสังเคราะห์** (ร่างโดย Claude, 5 ก.ค. 2026) จำลองสถานการณ์นโยบาย "ค่าธรรมเนียมรถติด กทม." เพื่อใช้ทดสอบ ingestion / knowledge graph / simulation เท่านั้น — **ไม่ใช่ข่าวจริง** ตัวเลขและเหตุการณ์ทั้งหมดสมมติขึ้น

หลักการที่ใช้ร่าง:
- ไม่มี PII: ไม่ระบุชื่อบุคคลใดๆ ใช้ตำแหน่ง/บทบาทแทน (สอดคล้อง GOV-01)
- ครอบคลุมหลายมุมมอง (สนับสนุน/ค้าน/ตั้งเงื่อนไข) และหลายช่องทางสื่อสาร (ข่าว, โพล, โซเชียล, ปากต่อปาก) เพื่อให้ persona factory และ social fabric มีวัตถุดิบหลากหลาย
- สอดคล้องกับ User Story US-1 ใน PRD (ทีมนโยบาย กทม. ซ้อมปฏิกิริยาต่อมาตรการเก็บค่าธรรมเนียมรถติด)

เมื่อมี corpus ข่าวจริง ให้แทนที่หรือเพิ่มไฟล์ตามรูปแบบชื่อเดียวกัน
